ECHO'S ANGELS FINAL WEBSITE PACKAGE
==================================

HOW TO OPEN IT
--------------
1. Unzip the folder.
2. On Windows, double-click: RUN_WEBSITE_WINDOWS.bat
3. Your browser should open to http://localhost:8080
4. If that does not work, open START_HERE.html or index.html.

ADMIN EDIT MODE
---------------
Click Settings on the site.
Default password: echo

Admin can edit business info, homepage text, services, gallery, FAQ, social links, section visibility, images, and SEO.

IMPORTANT ADMIN NOTE
--------------------
Because this is a free static website, Admin edits are first saved as a browser draft.
To make edits permanent for the live website:
1. Click Settings
2. Make edits
3. Click Save Draft
4. Click Export content.json
5. Replace the website's content.json file with the exported one
6. Upload/commit that new content.json to GitHub Pages

PHOTO WORKFLOW
--------------
Put real photos inside: assets/images/
Then use paths like: assets/images/my-dog-photo.jpg

Recommended image sizes:
- Hero/slideshow: 1600px wide, under 1 MB if possible
- Gallery/services: 800-1200px wide
- Use .jpg or .webp for photos

BEFORE PUBLIC LAUNCH
--------------------
- Replace placeholder photos with real photos
- Add social media links in Settings
- Add phone number only if you want it public
- Update testimonials or disable that section until you have real ones
- Consider setting up Formspree for a real contact form backend
- Publish to GitHub Pages
- Buy/connect EchosAngels.com later

DOMAIN NOTE
-----------
You do NOT need to buy the domain before building or publishing.
You can launch first on a free GitHub Pages URL, then connect www.EchosAngels.com later.
